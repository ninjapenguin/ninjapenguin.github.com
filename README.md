# ninjapenguin.co.uk

Static archive of the old Octopress-powered ninjapenguin.co.uk site.

This repository is the GitHub Pages publishing root. The site is served directly
from the files in this repository, so the old Ruby, Sinatra, and Heroku runtime
are no longer required.

The previous Tapir search integration, RSS feed, and external sidebar widgets
have been removed.

GitHub Pages serves `ninjapenguin.co.uk` from `master` at the repository root.
