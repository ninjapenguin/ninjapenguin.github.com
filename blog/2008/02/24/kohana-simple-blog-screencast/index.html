Kohana Simple Blog Screencast

Published on: Feb 24th, 2008

So following on from yesterdays screencast I have put together another short tutorial showing the first part of a simple blog application.

I had some feedback yesterday on the sound levels of the first screencast so I have upped the levels on this one, I have also increased the quality slightly, although this has also increased the file size (this screencast is 30mb when zipped with 13minute length)

I will only offer this one as compressed download (due to the bigger size) so any feedback would be greatly received!
